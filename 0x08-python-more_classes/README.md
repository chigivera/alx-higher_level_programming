# Python - More Classes and Objects

This project contains multiple implementations of a Rectangle class, each building upon the previous one with additional features. Below is a description of each file and its corresponding task.

## Files and Descriptions

### 0-rectangle.py
- Implements an empty class `Rectangle` that defines a rectangle.

### 1-rectangle.py
- Builds on 0-rectangle.py.
- Adds private instance attributes `width` and `height` with property getters and setters.

### 2-rectangle.py
- Builds on 1-rectangle.py.
- Adds public instance methods `area()` and `perimeter()`.

### 3-rectangle.py
- Builds on 2-rectangle.py.
- Adds string representation for `print()` and `str()`.

### 4-rectangle.py
- Builds on 3-rectangle.py.
- Adds string representation for `repr()`.

### 5-rectangle.py
- Builds on 4-rectangle.py.
- Prints a message when an instance is deleted.

### 6-rectangle.py
- Builds on 5-rectangle.py.
- Adds a public class attribute `number_of_instances`.

### 7-rectangle.py
- Builds on 6-rectangle.py.
- Adds a public class attribute `print_symbol` for string representation.

### 8-rectangle.py
- Builds on 7-rectangle.py.
- Adds a static method `bigger_or_equal()` to compare rectangles.

### 9-rectangle.py
- Builds on 8-rectangle.py.
- Adds a class method `square()` to create a square Rectangle instance.

### 101-nqueens.py
- Solves the N queens problem.

## Requirements
- All files interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.8.5)
- All files should end with a new line
- The first line of all files should be exactly `#!/usr/bin/python3`
- Code should use pycodestyle (version 2.8.*)
- All files must be executable
